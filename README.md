# IT1214-Student-Grade-System

## Project Purpose
This project is a Java Console-Based Student Grade Management System developed using Object-Oriented Programming (OOP) concepts.

The system allows teachers to:
- Add student records
- Display all students
- Search students by ID
- Calculate average marks

## How to Run

1. Open the project in IntelliJ IDEA or NetBeans.
2. Compile all Java files.
3. Run the StudentGradeSystem class.
4. Use the menu options to manage student records.
